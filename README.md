# BATMAN - The Missing Black Cat

A website to help find a missing black cat named Batman in Taman Antarabangsa, Likas.
This project includes a 3D mini-game, clues, real photos, and contact information.

## Features

- **3D Mini-Game**: Help Batman find his way home (built with Three.js).
- **Multi-language Support**: English, Bahasa Melayu, and Chinese (Switchable).
- **Interactive Story**: Scroll-triggered animations using GSAP.
- **Responsive Design**: Works on Desktop and Mobile.
- **Live Status**: Simulated CCTV surveillance status bar.

## Technologies

- HTML5 / CSS3
- JavaScript (Vanilla)
- [Three.js](https://threejs.org/) (3D Graphics)
- [GSAP](https://greensock.com/gsap/) (Animations)

## Deployment

This is a static website and can be deployed easily to **Vercel**, **Netlify**, or **GitHub Pages**.

### Deploy to Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in this directory.
3. Follow the prompts.

Or push to GitHub and import the project in the Vercel Dashboard.
